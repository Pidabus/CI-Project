#GradeMinds: Student Academic Performance Prediction and Risk Classification

## 📖 Project Overview
Academic underperformance often goes undetected until formative assessment outcomes are irreversible. This project proposes a hybrid computational intelligence framework designed to facilitate early pedagogical intervention. By synthesizing the high-dimensional pattern recognition of a Multilayer Perceptron (MLP) Neural Network with the interpretability of a Mamdani Fuzzy Inference System, this pipeline predicts student final grades and classifies them into actionable risk categories.

## 🏗️ System Architecture
The project is divided into a three-stage pipeline managed by a central integration script.
